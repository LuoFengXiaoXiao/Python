# TMS切片转换工具
## 描述：
> <br> Cesiumlab转换的TMS切片做了简单的加密，所以需要进行png的名称修改

## 使用方法：
> 将下载的文件放在tilemapresource.xml文件存在的文件夹下，然后运行即可。
